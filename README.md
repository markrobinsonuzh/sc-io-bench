# sc-io-bench

This is a small benchmark just to understand I/O for single cell data in R and Python

## Overview

This benchmark was created using [OmniBenchmark](https://omnibenchmark.org) v0.4.0.post56+g6f8455fc7, a framework for automated scientific benchmarking.

## Getting Started

1. Install OmniBenchmark:
   ```bash
   pip install omnibenchmark
   ```

2. Run the benchmark:
   ```bash
   ob run benchmark.yaml
   ```
